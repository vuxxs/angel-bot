# Drebin Bot

Drebin Bot is an everything-you-need Typescript bot, designed for easily adding anything additional that you might need.

## Features

- Message and command interaction support
- Extensible command interface
- Built-in commands like purge, remind, poll, and more
- Easy integration with custom commands
- Deno-first runtime and test workflow

## Installation

Follow these steps to get the bot running on your local machine:

1. **Clone the repository**

```bash
git clone https://github.com/vuxxs/drebin-bot.git
```

2. **Enter the project**

```
cd drebin-bot
```

3. **Setup configuration** Copy `.env.example` to `.env` and fill in the environmental variables.

4. **Run the Bot**

```bash
deno task start
```

5. **Run automated tests**

```bash
deno task test
```

That's it! Your bot should be now running on your server.
